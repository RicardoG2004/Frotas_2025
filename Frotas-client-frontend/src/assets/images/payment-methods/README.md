# Imagens dos Métodos de Pagamento

Coloque aqui as imagens dos métodos de pagamento.

## Estrutura de Arquivos

Coloque as imagens com os seguintes nomes:

- `transferencia.png` - Transferência Bancária
- `debito-direto.png` - Débito Direto
- `mbway.png` - MB Way
- `multibanco.png` - Multibanco
- `cartao-debito.png` - Cartão de Débito
- `cartao-credito.png` - Cartão de Crédito
- `paypal.png` - PayPal
- `apple-pay.png` - Apple Pay
- `google-pay.png` - Google Pay
- `dinheiro.png` - Dinheiro
- `cheque.png` - Cheque
- `outro.png` - Outro

## Recomendações

- **Formato**: PNG ou SVG (transparente preferencialmente)
- **Tamanho**: Recomendado 200x200px ou maior (será redimensionado automaticamente)
- **Fundo**: Transparente ou branco
- **Qualidade**: Alta resolução para melhor visualização

## Exemplo de Uso

As imagens serão carregadas automaticamente pelo código. Certifique-se de que os nomes dos arquivos correspondem exatamente aos nomes especificados acima.
